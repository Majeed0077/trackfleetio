KG-27 Specification
The KG-27 is a high-performance 4G vehicle tracker
designed for advanced fleet and industrial applications.
It features a 9-90 VDC input range, versatile I/O
interfaces, and supports Lora sensor connectivity, with
robust security including remote cut-off and intelligent
status alerts.
This product supports:
1. 2G + 4G Cat.1
2. GNSS: GPS/ GALILEO/ BeiDou/ GLONASS
3. LoRa
4. Abundant External Interfaces (1-WIRE, RS485,
RS232, ADC)
5. 9-90 VDC Wide Operating Voltage
6. 6,000 Blind Spot Data Storage Entries
7. Configurator and FOTA WEB
